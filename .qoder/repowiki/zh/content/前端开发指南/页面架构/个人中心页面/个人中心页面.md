## 简介

畅搭(FreeDress)应用的个人中心页面组是用户账户管理的核心模块，包含三个主要页面：ProfileScreen个人资料页、EditProfileScreen编辑资料页和FavoritesScreen收藏夹页。该模块实现了完整的用户资料管理功能，包括个人信息展示、编辑权限控制和数据更新机制；提供了收藏夹功能，支持收藏物品展示、分类管理和批量操作；集成了用户统计功能，涵盖衣橱统计、搭配历史和使用数据分析。

## 故障排除指南

### 常见问题及解决方案

#### 登录状态异常

当用户登录状态异常时，可通过以下步骤排查：

1. **检查本地存储**
   - 验证ACCESS_TOKEN和REFRESH_TOKEN是否存在
   - 确认USER_INFO格式正确且可解析

2. **网络请求验证**
   - 检查API端点可达性
   - 验证认证令牌有效性

3. **状态同步**
   - 确保useAuthStore状态与本地存储一致
   - 检查异步操作的错误处理

#### 图像上传失败

图像上传失败的常见原因和解决方法：

```mermaid
flowchart TD
UploadError[上传失败] --> CheckNetwork["检查网络连接"]
CheckNetwork --> NetworkOK{"网络正常?"}
NetworkOK --> |否| FixNetwork["修复网络连接"]
NetworkOK --> |是| CheckImageSize["检查图像大小"]
CheckImageSize --> SizeOK{"图像小于限制?"}
SizeOK --> |否| ReduceSize["压缩图像"]
SizeOK --> |是| CheckFormat["检查图像格式"]
CheckFormat --> FormatOK{"格式支持?"}
FormatOK --> |否| ConvertFormat["转换图像格式"]
FormatOK --> |是| RetryUpload["重试上传"]
FixNetwork --> RetryUpload
ReduceSize --> RetryUpload
ConvertFormat --> RetryUpload
```

#### 列表数据加载问题

收藏列表数据加载异常的诊断流程：

```mermaid
flowchart TD
LoadError[数据加载失败] --> CheckAPI["检查API响应"]
CheckAPI --> APIOK{"API返回成功?"}
APIOK --> |否| CheckAuth["检查认证状态"]
APIOK --> |是| CheckData["检查数据格式"]
CheckAuth --> AuthOK{"认证有效?"}
AuthOK --> |否| ClearAuth["清除认证状态"]
AuthOK --> |是| CheckData
CheckData --> DataOK{"数据格式正确?"}
DataOK --> |否| TransformData["转换数据格式"]
DataOK --> |是| ShowEmpty["显示空状态"]
ClearAuth --> RetryLoad["重试加载"]
TransformData --> RetryLoad
ShowEmpty --> RetryLoad
```
