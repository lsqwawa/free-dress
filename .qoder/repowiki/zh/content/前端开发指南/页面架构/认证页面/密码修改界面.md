## 简介

密码修改界面是 FreeDress 应用中的一个关键安全功能模块，允许已登录用户修改其账户密码。该界面采用现代化的设计语言，结合了前端 React Native 技术栈和 NestJS 后端架构，提供了完整的密码修改流程，包括客户端验证、服务器端验证和安全的密码更新机制。

**更新** 新的认证系统引入了JWT令牌管理，增强了密码修改功能的安全性和可靠性。

## 认证系统详解

### JWT令牌管理

新的认证系统引入了完整的JWT令牌管理机制，确保密码修改功能的安全性：

#### 令牌生成流程

```mermaid
flowchart TD
Start([用户登录]) --> GeneratePayload[生成JWT载荷]
GeneratePayload --> SignAccess[签名访问令牌]
SignAccess --> SignRefresh[签名刷新令牌]
SignRefresh --> ReturnTokens[返回令牌对]
ReturnTokens --> StoreLocal[本地存储]
StoreLocal --> UseAPI[API调用]
UseAPI --> VerifyToken{验证令牌有效性}
VerifyToken --> |有效| AccessAPI[访问受保护API]
VerifyToken --> |无效| RefreshToken[刷新令牌]
RefreshToken --> StoreNew[存储新令牌]
StoreNew --> RetryAPI[重试API调用]
```

3. **权限控制**: 确保只有认证用户才能访问
4. **错误处理**: 统一的未授权异常处理

## 故障排除指南

### 常见问题及解决方案

#### 前端问题

1. **输入验证失败**
   - 检查密码长度是否符合要求
   - 确认新密码和确认密码是否一致
   - 验证旧密码是否正确

2. **API调用失败**
   - 检查网络连接状态
   - 验证JWT令牌的有效性
   - 查看服务器响应状态码

3. **界面显示异常**
   - 检查组件的props传递
   - 验证样式类名的正确性
   - 确认主题配置的完整性

4. **令牌刷新失败**
   - 检查刷新令牌是否过期
   - 验证本地存储中的令牌状态
   - 查看网络请求的响应

#### 后端问题

1. **密码修改失败**
   - 检查用户是否存在
   - 验证旧密码的正确性
   - 确认数据库连接正常

2. **安全验证失败**
   - 检查JWT配置
   - 验证用户权限
   - 查看日志记录

3. **认证接口异常**
   - 检查JWT密钥配置
   - 验证令牌格式和有效期
   - 确认Passport策略配置
